# 本會受邀至清大「勞動有查事」講座錄影
> 張貼者：2015年12月4日 下午12:38 Ming Che Lin   [ 已更新 2015年12月4日 下午12:40 ]

本會於11月25日應邀於清華大學「勞動有查事」講座演講，分享主題是「科技業的勞動現況與工會— 園區、輪班星人與科技人的勞動權益」。感謝清大同學的邀請，以及當天的熱烈提問。以下是當天活動錄影，投影片可於本頁面附件下載。
