## 會費繳交方式

### 會費收費標準

- 入會費 1000 元，於申請入會時繳交。

- 經常會費每月 300 元，可月繳、季繳、年繳，以預繳為原則。

- 會員遇失業、進修等導致暫無收入，或因重大事故導致經濟困難者，可向理事會提出申請，由理事會決議減繳或緩繳會費。

### 繳費方式

#### 一、線上繳費

可以[設定信用卡定期定額繳費](https://tueeit.neticrm.tw/civicrm/contribute/transact?reset=1&id=4)（最推薦，只要設定一次即可），或[透過 ATM 轉帳繳費](https://tueeit.neticrm.tw/civicrm/contribute/transact?reset=1&id=6)。

#### 二、至金融機構匯款或轉帳

會員須自行負擔金融機構可能收取的手續費。
匯款完成後，請[電郵](tueeit@tueeit.org.tw)告知姓名、轉帳時間、轉帳金額、帳號末五碼。

```
玉山銀行（808）六家分行
帳號：1377-940-159230
戶名：台灣電子電機資訊產業工會
```

#### 三、現金支付

會員可親自到工會辦公室使用現金繳費。

```
繳款時間：星期一至五，10:00-18:00
繳款地點：新竹縣湖口鄉大智路 42 號
```
