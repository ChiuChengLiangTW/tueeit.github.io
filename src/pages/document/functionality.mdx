# 工會有什麼作用？

如果有了工會，而且工會確實發生作用，就可以：

- 每年調薪談判和年終獎金談判，用集體的力量跟公司談，不是一切公司說了算。

- 公司要改變薪資、工時、分紅等制度，無薪休假要先經工會同意，責任制的工作時間認定由勞資協商。

- 可以加強參與職工福利委員會，督導公司提撥金額，讓福利會更符合員工需求。

- 監督勞工退休準備金提撥，避免公司垮掉，工人拿不到資遣費、退休金。

- 工人被欺負，或升遷、考績、調薪不公，可向工會申訴，由工會代表和公司談；工會還可以找勞服中心、產業總工會協助，不會求助無門，或因為不懂法規而被唬弄。

- 簽訂團體協約，作為勞資關係的憲法，把一些優於勞基法的制度確定下來。

- 公司要裁員，由工會去協商、開條件，不是今天把我叫進辦公室，就說明天不用來了。關廠或併購，工人怎麼辦，也是工會先去談。

- 幫員工講話，可能會被修理；以工會名義來為員工說話，法律上多一層保護。

- 公司違法，去檢舉還要具名，來查了不一定會罰，罰了公司不一定會痛（罰款都很低），或者叫工人自己去打個三年五年的官司；以工會出面具名檢舉比較不怕，必要時可以進行勞資爭議。

- 用工會集體的名義向政府抗議勞動法規修惡、要求工人福利，人多一起來，政府才會重視。
